# Puppteer를 활용한 공모전 크롤링 사이트
</br>

## Previews

- 크롤링한 공모전 자료들을 제공해주는 웹 사이트
    - 최신 공모전 자료들을 열람할 수 있습니다
    - 로그인을 진행하여, 맘에 드는 공모전 자료들을 관리할 수 있습니다
    - 댓글 및 커뮤니티 기능을 통해 공모전의 대해 토론할 수 있습니다
