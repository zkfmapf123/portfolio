# 개미처럼 공부해서 수능을 작살내자
</br>

- 수능 및 공부관리 모바일 App
    - 할일 목록을 통한 공부 스케쥴을 관리할 수 있다.
    - 공부 타이머를 활용하여 실용적인 공부 스케쥴을 관리한다.
    - 그날 그날 공부한 날의 데이터를 합산하여 보여준다.
    - 일주일간의 통계를 시각적으로 보여준다.
